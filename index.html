jQuery Tutorial

What is jquery?

It's a fast, small, and feature-rich javaScript library.

jQuery important features:

  • DOM selection:
    •    jQuery provides selector to retrieve DOM element based on different criteria like tag name, id, css class name, attribute name, value, nth child in hierarchy ... etc.
  • DOM manipulation:
    •    It enables manipulating DOM, i.e, adding, removing elements from the DOM, modifying HTML content, CSS class etc.
  • Special effects:
    •    It enables applying special effects to the DOM elements like showing or hiding elements, fade-in or fade-out of visibility, sliding effects, animation etc.
  • Events:
    •    It includes functions that are equivalent to DOM events like click, dblclick, mouseenter, mouseover,....
  • AJAX:
    •    it includes easy to use AJAX functions to load data from servers without reloading whole page.
  • Cross-browser support:
    •    It handles the cross-browser issues.

Advantages of jQuery:

  • Easy to learn:
    •    jQuery is easy to learn, since it supports javaScript style coding.
  • Write less Do more:
    •    it provides a rich set of features that increase developers' productivity by writin less and readable code.
  • Excellent API documentaion:
    •    it provides excellent online API documentaion.
  • Cross-browser support:
    •    It supposts Cross-browser without writing extra code.
  • Unobtrusive:
    •    It allows seperation of concerns by seperating HTML and jQuery code.

Notes:

    • The difference between $(document).ready() and window.onload(): is that the $(document).ready() determines when the full DOM hierarchy is loaded whereas window.onload() event is raised whin the entire window is loaded including DOM, images, CSS and other required resources.
    • The DOM loads before the entire window loads.
    • $(document).ready() doesn't wait for images to be loaded, it gets called as soon as DOM hierarchy is constructed fully.
