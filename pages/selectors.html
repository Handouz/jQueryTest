jQuery selectors

jQuery selectors enables you to find DOM elements in your web page.

Most of the times you will start with selector function $() in the jQuery

jQuery syntax:

$(selector expression, context)

The context parameter is optional, it specifies elements in a DOM hierarchy from where jQuery starts searching for matching elements.

Select elements by Name:

To select element by name just use the name of the element such as the following example:

$('p')

Here we selected all the p elements in the DOM

Select element by Id:

To select element by id just use the #Id_Of_The_Element (It's a bad practice).

$('#id'):

Selects the element with a specific id in the DOM

Select element by Attribute:

jQuery allows to select element based on a specific attribute, just put the attribute in a square brackets []

$('[target]')

Just select the elements that has the attribute (target).

We can also specify a value of an attribute in attribute selector.

For example: 

$('[target="_blank"]')
it selects the elements that has an attribute (target) with value (_blank)
