jQuery selectors

jQuery selectors enables you to find DOM elements in your web page.

Most of the times you will start with selector function $() in the jQuery

jQuery syntax:

$(selector expression, context)

The context parameter is optional, it specifies elements in a DOM hierarchy from where jQuery starts searching for matching elements.

Select elements by Name:

To select element by name just use the name of the element such as the following example:

$('p')

Here we selected all the p elements in the DOM
